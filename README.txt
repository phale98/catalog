Udacity Catalog App:
This project runs a server for a catalog style app.

Getting Started:
		To get started unzip the file in the directory of your choosing.
	Prerequisites:
		Must have python, sqlalchemy, and flask installed. sqlite3 command line tool
		is also reccomended if you intend to view the database directly.
	Installing:
		Unzip the project into the target directory and run the catalogApp.py file to
		to start the server.
	Before You Run:
		Before the main project will run, you must run database_setup.py. In addition,
		please replace the Google API keys with your own. 
	
Built With:
	SQLAlchemy, Flask, and Python.
